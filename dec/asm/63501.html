63501: Restart the command list unless it's time for assembly
Used by command list 72. Restarts the command list (making MR WACKER go back up to his study) unless it's time to start assembly.
Input
H 200 (MR WACKER)
63501 LD A,(32740) A=MSB of the lesson clock (which starts off at 16)
63504 CP 9 Is it time to start assembly yet?
63506 JP NC,61624 Restart the command list if not
63509 JP 25484 Otherwise move to the next command in the command list
