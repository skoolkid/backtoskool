D43C: MSBs of addresses of ERIC-handling routines
Used by the routine at F7AD. The corresponding LSBs can be found at D33C.
D43C DEFB $F2 MSB of F28E (used when bit 7 is set at 7FFB)
D43D DEFB $00 Unused (when bit 6 is set at 7FFB, ERIC is incapacitated)
D43E DEFB $5E MSB of 5EAD (used when bit 5 is set at 7FFB)
D43F DEFB $5E MSB of 5E6F (used when bit 4 is set at 7FFB)
D440 DEFB $E1 MSB of E145 (used when bit 3 is set at 7FFB)
D441 DEFB $F2 MSB of F2E2 (used when bit 2 is set at 7FFB)
D442 DEFB $00 Unused (when bit 1 is set at 7FFB, ERIC's secondary status flags at 7FED are checked)
D443 DEFB $5D MSB of 5D63 (used when bit 0 is set at 7FFB)
