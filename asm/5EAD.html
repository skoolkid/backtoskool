5EAD: Deal with ERIC when he's firing the catapult, hitting or kissing
This routine is called by the routine at F7AD when bit 5 at 7FFB is set (by the routine at 5EB7).
Input
Output
5EAD LD HL,$7FF3 7FF3 holds ERIC's main action timer
5EB0 DEC (HL) Is it time to deal with ERIC yet?
5EB1 RET NZ Return if not
5EB2 LD (HL),$05 Reset the main action timer to 5
5EB4 JP $E14F Jump to the routine address stored in 7FD7, which will deal with the next phase of ERIC's animation
