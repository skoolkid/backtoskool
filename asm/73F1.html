73F1: Open the Science Lab storeroom door if ERIC has the key
Used by the routine at 6E9B. Opens the Science Lab storeroom door if ERIC is facing it, has the key, and is trying to move forwards.
Input
H 0x7F
Output
73F1 LD L,$EB HL=7FEB (inventory flags)
73F3 BIT 1,(HL) Has ERIC got the storeroom key?
73F5 RET Z Return if not
73F6 LD B,$01 1=open (as opposed to 0=close)
73F8 LD A,$04 This is the Science Lab storeroom door identifier
73FA JP $7040 Open the storeroom door
